## v1.5 — 增强版（中期）

### 🤖 AI 升级
- [ ] **AI 记忆系统**：AI 记住之前轮次的发言和投票，形成一致性人格
- [ ] **AI 女巫推理**：基于发言频率、投票一致性等线索推断狼人
- [ ] **AI 狼人协作**：狼人之间通过微妙的发言暗示配合
- [ ] **AI 预言家策略**：学会在适当时机"跳身份"（报查验）
- [ ] **多模型支持**：支持 OpenAI / Claude / 本地模型

### 🎮 游戏性
- [ ] **遗言系统**：被投票出局的玩家发表遗言
- [ ] **女巫不可自救规则选项**
- [ ] **配置面板**：自定义人数（6/9/12人）、角色组合
- [ ] **守卫角色**：可选加入，完善角色阵容
- [ ] **白痴/长老角色**：更多角色选择

### 📊 统计
- [ ] **战绩记录**：胜负统计、角色胜率（localStorage）
- [ ] **复盘回放**：保存每局发言和行动记录

## v2.0 — 完整版（长期）

### 🌐 多人联机
- [ ] **WebSocket 房间系统**：创建/加入房间
- [ ] **实时同步**：发言、投票实时广播
- [ ] **语音通话**：WebRTC 语音聊天（替代 Web Speech API）
- [ ] **旁观模式**：观众可以看直播
- [ ] **房间密码**：私密房间

### 🎨 视觉升级
- [ ] **角色立绘**：每个角色有独特插画
- [ ] **夜晚动画**：月落日出过渡动画
- [ ] **击杀动画**：狼人猎杀特效
- [ ] **音效系统**：背景音乐 + 音效（夜晚狼嚎、投票敲锤）
- [ ] **移动端适配**：响应式布局

### 🧠 AI 2.0
- [ ] **LLM 驱动的夜晚决策**：不只发言，夜晚行动也用 LLM 判断
- [ ] **AI 性格系统**：每种性格有独特的行为模式（不只是语音参数）
- [ ] **AI 学习**：从历史对局中学习更优策略
- [ ] **难度选择**：简单/普通/困难（影响 AI 推理准确性）

### 📱 平台化
- [ ] **PWA 支持**：可安装到手机桌面，离线可玩（无 API 时用模板发言）
- [ ] **多语言**：英文版本
- [ ] **分享对局**：生成对局摘要图片分享到社交媒体
